stars Helpful Links


  • settings_applicationsCAD and Rendering

    • AutoDesk Inventor is the software we use to create our final robot assemblies. It is a relatively easy to use software that can be imported into AutoDesk Showcase, which we wuse to create custom renders and animations.
      • Inventor Educational is free to students and teams.
      • Inventor Trenches is a great blog with links to easy to understand tutorials

    • PTC Creo is another great CAD software which we use to create our custom parts for 3-D Printing
      • PTC Creo is free to FIRST teams and has a great community there as well.
      • PTC Learning Exchange with great tutorials and answers to common problems with PTC Creo Parametric

    • Blender is great for creating animations and high quality renders of your robot and is completely free. However, if you are just using it for robot renders, we suggest sticking with Autodesk Showcase because it has a much lower learning curve and imports Inventor files better.
      • Blender's Website has some great tutorials on almost every aspect of the software

  • buildHardware

    • Parts Suppliers are great places to order parts beyond the basic Tetrix parts. This year, we are making the push towards increasingly minimal Tetrix because other custom build systems allow us to make better, stronger, and cooler robots. Here are some of our favorites:
      • ServoCity is a great place to get parts for all kinds of motion. (i.e. Bearings, Hubs, Axles, etc) They are a little expensive but offer a 25% discount to all FIRST teams!
      • AndyMark is a great supplier for FIRST teama. They offer the field kits and all kinds of wheels, gears, and sprockets that can be used in FTC. They are also the current distributor for REV Robotics Extrusion.
      • REV Robotics is an amazing new build system made by two pretty cool guys that have been involved with FIRST teams for more than 20 years. The cornerstone is very low cost aluminum extrusion that is extremely strong and versatile. It is currently being distributed through AndyMark.
      • OpenBuilds is the supplier for V-Slot aluminum extrusion. This is a great system for linear motion with its high quality bearing rollers. We recommend the Mini V Wheel and plate system because of its strength and small size. We have used it for two years now and will be releasing a tutorial soon.
      • McMaster-Carr is a great supplier for all kinds of metal products. We use them mostly for aluminum stock and fasteners (nuts, bolts, screws, etc.)

    • 3D Printers allow for immense customization and are a great resource for FIRST teams.
      • Ekocycle is a new 3d printer that uses recycled plastic to print with. Last year, FIRST partnered with Cubify to provide Ekocycle printers to FTC teams that applied for the grant. Hopefully, they'll be do the same thing again this year.
      • MakerBot is a great and affordable desktop 3d printer that can print with PLA or ABS plastic. If you are willing to spend the money, we suggest investing in one of these.

  • codeSoftware

    • Android Studio is the main IDE that will be used to program your robot, starting this season. The new platform, developed by Modern Robotics Inc. is Java based and uses two Android devices to remotely control the robot.
      • Download Android sutdio here. It will walk you through the setup process and it is also completely free.
      • Intellitek Education is a great wesbite with step by step walkthroughs of programming with the new platform.
      • The FTC Forum is a great resource for discussing the new platform. Everyone is figuring it out together since it is new.

    • ftc_app is the Github repository for the Robot Controller app and is updated every few weeks with tweaks, driver updates, and more Documentation.
      • Here is the main repository.
      • The Documentation is very helpful. The tutorial folder contains basic tutorial pdf files. "javadoc/index.html" taks you to the web portal for the documentation. (You must download the Repo to access this part!)

  • whatshotCompetition Day

    • Scouting is essential to a competition. The more people that do good scouting, the better the chance there is that the best teams will advance.
      • FTC Mobile Scouter is an Android app developed by a member of our sister team, 6299 QuadX that is useful for inputing match data and exporting it for analysis on your computer.
      • SCOUTing is an Android app developed by team 524 Bossbots and is one of our favorite apps for scouting teams. It allows for a picture of the teams robot and custom tags that can be used to filter teams. It also includes a scoring feature for this years game.
      • FTC Match Analyzer is a Java application created by us for use in ranking teams by match performance. The algorithm is constantly updated based on that years game and how team are perfoming at different levels. For more info check out the Readme.md.

  • publicCommunities

    • FTC Tec Network is a coalition of over 100 FTC teams from 8 different countries around the world. It is mainly a Google+ community with teams that host online video Hangouts over a variety of topics.
      • Here is their main website. You have to request to join at this link.

    • Reddit has a relatively large FTC community with several posts a day. Very useful if you need help woth something or want to discuss anything FTC with other teams.
      • r/FTC is the FTC subreddit.

    • The FTC Forum is another great resource. It has rule clarifications and team discussions.
      • Official FTC Forum
